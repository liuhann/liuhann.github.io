The basic table structure is like this 


    <div class="ultratable">
        <div class="ultra-header"></div>
        <div class="ultra-body">
            <div class="ultra-body-content">
                <div class="ultra-row">
                <div class="ultra-cell"><a class="download.html">Download&amp;Install</a></div>
                </div>
            </div>
        </div>
        <div class="ultra-footer"></div>
    </div>

If you are using ultra.css, the div.ultra-cell style is set to 


    display: table-cell;

This is a html style for a pure table. If your table is more like a list, you can use your own styles
